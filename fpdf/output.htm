Output

string Output([string dest [, string name [, boolean isUTF8]]])

Descripció

Envia el document a la destinació donada: cadena, fitxer local o navegador. En aquest cas, es pot usar el connector (si hi és) o es pot forçar una descàrrega (quadre de diàleg "Desar com").
El mètode primer executa Close() si cal finalitzar el document.

Paràmetres

dest
Destinació on enviar-hi el document. Pot prendre un dels següents valors:
  • I: envia el fitxer inserit al navegador. S'usa el connector si està disponible.
  • D: envia al navegador i força una descàrrega de fitxer amb el nom donat per name.
  • F: desa a un fitxer local amb el nom donat per name.
  • S: torna el document com una cadena.
El valor predeterminat és I.
name
El nom del fitxer. S'ignora si el destí és S.
El valor predeterminat és doc.pdf.
isUTF8
Indica si el nom està codificat en ISO-8859-1 (false) o UTF-8 (true). Només s'usa per a destins I i D.
El valor predeterminat és false.

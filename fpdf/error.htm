Error

Error(string msg)

Descripció

Aquest mètode es crida automàticament si hi ha un error fatal; llença simplement una excepció amb el missatge indicat.
Una classe heretada pot ignorar-lo per a personalitzar el control d'errors però pot ser que no surti mai, a més que el document resultant serà defectuós probablement.

Paràmetres

msg
El missatge d'error.
